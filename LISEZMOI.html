Pré-requis

Les seuls prérequis sont que:

  • un compilateur java doit être sur le chemin d'accès ("path");
  • l'outil Maven doit être sur le chemin d'accès ("path").

Compilation du programme

    Pour compiler le programme, ouvrir un shell, se déplace à la racine du dossier puis exécuter mvn compile

Exécution du programme

    Pour exécuter le programme, ouvrir un shell, se déplace à la racine du dossier puis exécuter mvn exec:java

Exécution les tests du programme

    Pour exécuter les tests du programme, ouvrir un shell, se déplace à la racine du dossier puis exécuter mvn test
